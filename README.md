# vue-pocket48

基于 [Vue](https://vuejs.org/) + [Node](https://nodejs.org) 的 web 版口袋48，请谨慎登录

**目前无法查看口袋房间**

## 如何使用？

克隆本仓库后

   ```shell
   cd Vue-pocket48
   npm install
   npm run build
   node server.js
   ```

## 主要功能

* 成员房间消息
* 成员直播/录播
* 录播弹幕
* 公演直播/录播
* 各队成员列表 (含CKG48成员)
* 各成员详细信息
* 关注/取关成员房间（在成员详细信息页面关注）
* 成员房间聚聚留言
